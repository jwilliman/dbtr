# dbtr

<!-- badges: start -->
<!-- badges: end -->

The goal of dbtr is to provide a collection of R tools for working with
data generated by diabetes technologies including CGM sensors and pumps.
Primarily developed for research rather than individual use. Simplifies
API process for extracting data from Nightscout and creating summary
metrics. `dbtr` makes use of the `httr2` package for creating API calls
and retrieving data, the `data.table` package for fast data cleaning,
and the excellent `iglu` package for creating CGM summary metrics.

## Installation

You can install the development version of dbtr from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("jwilliman/dbtr")
```
